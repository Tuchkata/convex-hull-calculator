Shortcut key

Action
Ctrl + N  Create a new canvas and delete all points
Ctrl + O  Open existing file with point coordinates
Ctrl + S  Save the coordinates of the points drawn to the canvas
Ctrl + Shift + S  Save the coordinates to another file (Save as)
Ctrl + Z  Undo the last change
Ctrl + Y  Redo the last change
F1  Show the Help window
Alt + F4  Close the application
